# Qt-CoinGame
Qt 5 以上运行 debug版本，也可以自己打包release版
此游戏是翻金币，包含银币和金币，全部变为金币获胜，翻一个币周围币也会跟着反动。
Qt 大概涉及 UI,音效，资源添加，定时器，动画，画家，基本槽函数和一些自定义信号连接，事件的重载和自定义，自定义按钮封装调用等等 
此处用的是QT 5.32版运行，以上版本都已可以运行
